### 环境
1. 阿里云vpc专用网络环境，vps私有ip 192.168.0.0/16
2. linux centos 7.4 发行版,建议使用服务商镜像制作自定义镜像(yum update -y)
3. 仅支持linux 或 mac os 作为执行主机
4. 需修改host文件,填入阿里云各主机外网ip(绑定弹性网卡)
